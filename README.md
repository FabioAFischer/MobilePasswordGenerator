Ao iniciar a aplicação, o usuário é direcionado automaticamente para a tela de login (Sign in). Nessa tela, é necessário preencher os campos de e-mail e senha para que o botão “Entrar” seja habilitado. Após preencher os campos, ao clicar em “Entrar”, o usuário é redirecionado para a tela principal do sistema, onde está localizado o gerador de senhas. Caso o usuário ainda não possua uma conta, é possível acessar a tela de cadastro clicando na opção “Crie agora”.

Na tela de cadastro (Sign up), o usuário deve preencher todos os campos solicitados, incluindo nome, e-mail, senha e confirmação de senha. O botão “Registrar” só será habilitado quando todos os campos estiverem preenchidos corretamente e as senhas forem idênticas. Após realizar o cadastro, o usuário é redirecionado novamente para a tela de login, com o campo de e-mail já preenchido automaticamente.

Após o login, o usuário acessa a tela principal (Home), onde pode gerar uma senha aleatória ao clicar no botão “Gerar senha”. Após gerar a senha, novas ações são habilitadas, como copiar a senha ou salvá-la. Ao clicar em “Salvar”, uma janela (modal) é exibida, permitindo que o usuário informe o nome do aplicativo ou finalidade da senha. A senha gerada já aparece preenchida automaticamente e não pode ser editada. Ao confirmar a ação, os dados são armazenados localmente no dispositivo.

Na tela de histórico, o usuário pode visualizar todas as senhas salvas. Cada senha é exibida em formato de cartão, inicialmente oculta. O usuário pode visualizar ou esconder a senha ao clicar no ícone correspondente, copiar a senha com outro ícone, ou excluí-la permanentemente do dispositivo. Sempre que o usuário sair e retornar a essa tela, as senhas voltam a ficar ocultas por padrão.
